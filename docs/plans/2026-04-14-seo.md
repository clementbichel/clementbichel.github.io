# SEO Local Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Optimiser le référencement local de clementbichel.fr pour les requêtes "coach agile Bordeaux" / "coach produit Bordeaux".

**Architecture:** Modifications directes dans `index.html` (méta tags, JSON-LD, alt images) et création de deux fichiers à la racine (`robots.txt`, `sitemap.xml`). Aucun outil de build — site HTML vanilla.

**Tech Stack:** HTML5 vanilla, Schema.org JSON-LD, GitHub Pages

---

### Task 1 : Méta tags — title, description, canonical

**Files:**
- Modify: `index.html:3-8`

**Step 1 : Remplacer le `<title>` et la `<meta description>`**

Dans `index.html`, remplacer les lignes 4–6 :

```html
<meta name="description" content="Clément Bichel — Coach d'équipe et de managers à Bordeaux. +15 ans d'expérience en coaching agile et produit." />
<title>Clément Bichel — Coach agile à Bordeaux</title>
```

par :

```html
<meta name="description" content="Coach agile et coach produit freelance à Bordeaux. +15 ans d'expérience : SAFe, Scrum, coaching PM/PO, transformation agile. Première consultation offerte." />
<title>Coach agile &amp; produit à Bordeaux — Clément Bichel</title>
```

**Step 2 : Ajouter la balise canonical juste après**

Après le `<title>`, ajouter :

```html
<link rel="canonical" href="https://clementbichel.fr/" />
```

**Step 3 : Vérifier dans le navigateur**

```bash
open index.html
```

Inspecter le `<head>` dans DevTools (Cmd+Option+I → Elements) et vérifier que :
- `<title>` = "Coach agile & produit à Bordeaux — Clément Bichel"
- `<meta name="description">` contient "Bordeaux" et "SAFe"
- `<link rel="canonical">` est présent

**Step 4 : Commit**

```bash
git add index.html
git commit -m "seo: optimiser title, meta description et canonical"
```
