# Rafraîchissement visuel ciblé — Plan d'implémentation

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Rafraîchir l'identité visuelle du site (typographie, palette, CTA, nav, témoignages, hover) sans modifier la structure SEO ni la logique JS.

**Architecture:** Modifications dans `style.css` (polices, variables, nouveaux composants) et `index.html` (3 points d'insertion : Google Fonts, nav brand, CTA, details/summary témoignages). Aucun changement à `main.js`.

**Tech Stack:** HTML/CSS vanilla. Pas de build. Vérification visuelle en ouvrant `index.html` dans un navigateur (ou `python3 -m http.server 8080`).

### Task 1 : Importer les polices Google Fonts

**Files:**
- Modify: `index.html` — dans `<head>`, avant `<link rel="stylesheet" href="style.css" />`

**Step 1 : Ajouter les balises Google Fonts**

Dans `index.html`, insérer juste avant `<link rel="stylesheet" href="style.css" />` :

```html
  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
  <link href="https://fonts.googleapis.com/css2?family=Fraunces:ital,opsz,wght@0,9..144,400;0,9..144,600;0,9..144,700;1,9..144,600&family=DM+Sans:wght@400;500&display=swap" rel="stylesheet" />
```

**Step 2 : Vérifier dans le navigateur**

Ouvrir `index.html`. Ouvrir les DevTools → Network → filtrer "fonts.googleapis". Deux requêtes doivent apparaître.

**Step 3 : Commit**

```bash
git add index.html
git commit -m "style: importer Fraunces et DM Sans depuis Google Fonts"
```
